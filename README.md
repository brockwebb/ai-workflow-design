# AI Workflow Design for Official Statistics

Practical design patterns for building reliable AI-powered data processing pipelines in federal statistical agencies and research organizations.

**Status:** Early development. Private until ready for public release.

## What This Is

A short, open-source book on designing LLM/AI workflows for statistical research — classification, data cleaning, imputation, detection, and extraction. Covers ensemble methods, multi-model architectures, checkpoint/recovery patterns, cost engineering, and evaluation-by-design.

This is the sequel to [AI for Official Statistics](https://brockwebb.github.io/ai4stats/). Where that book teaches statisticians what AI is, this one teaches builders how to design AI workflows that work reliably.

## Who It's For

Data scientists and statisticians building AI-powered pipelines for:
- Survey question classification and coding
- Data cleaning and edit/imputation
- Entity extraction and pattern detection
- Disclosure review assistance
- Any statistical research workflow with an LLM component
